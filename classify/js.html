递归 #

函数自己调用自己，一定要有临界值，否则会栈溢出

递归函数求和

tsx
type Sum = (n: number) => number
const sum:Sum = (n: number): number => {
    if (n === 1) {
        return 1
    }
    return n + sum(n - 1)
}
console.log(sum(100)); // 5500

tsx

// 1 1 2 3 5 8 13
let fib: Fib = (index: number): number => {
    if (index <= 1) {
        return 1
    }
    return fib(index - 1) + fib(index - 2)
}
console.log(fib(5)); // 8

闭包 #

函数里面再定义函数，内部的函数可以访问外部函数的作用域变量，如果外部函数不暴露内部函数，外界就不知道内部函数的存在...形成一个私有的函数

求两个数的平方和

tsx
let squareSum = (a: number, b: number): number => {
    let square = (x: number): number => {
        return x * x
    }
    return square(a) + square(b)
}
console.log(squareSum(2, 6)); // 40

柯里化 #

将接收多个参数的函数变成接收一个参数的内部的函数，这样他们的值就能保留

计算三个整数之和

tsx
let addThreeNums = (a:number) => {
    return (b:number) => {
        return (c:number) => {
            return a + b + c
        }
    }
}
console.log(addThreeNums(1)(2)(3)); // 6
let fixedTwo = addThreeNums(1)(2) // 将前两次返回值固定
console.log(fixedTwo(4)); // 7
console.log(fixedTwo(5))；// 8

值传递和引用传递 #

在函数的参数中，数组和对象是按引用来传递的，也就是说传递的是内存地址.当修改他们的值之后，所有用到它地址的值都会变化...基本类型是按值来传递的，他们的值会赋值一份新的，在函数中修改他们的值不会影响它之前的值

引用传递

js
let arr = [1,2,3]
function byReference(arr) {
    arr[0] = 5
    console.log(arr); // [5,2,3]
}
byReference(arr);
console.log(arr);// [5,2,3]

逻辑 => |(或)

界定符 => ^:匹配字符开头 &:匹配字符结尾

js
console.log(str.match(/[a-z]/g)); // 匹配a-z区间中的任意一个字符
console.log(str.match(/[^0-9]/g)); // 匹配除0-9区间外的任意一个字符
console.log(str.match(/This|contains/g)); // ['This', 'contains']

分组

使用()对模式进行分组，分组的内容会当成一个整体，并且执行的结果会把分组的内容返回回来

js
var str = 'this that this and that'

var str = 'aaaab abb cddaa'
console.log(str.match(/(aa){2}/g)); //aa出现两次 => ['aaaa']

Set

Set对象是一个没有重复元素的集合，可以对元素进行增删查、遍历

增:set.add(value)

删:set.delete(value)

查:set.has(value) => false||true

清空:set.clear()

遍历:set.forEach(value => console.log(value)) console.log(...set.keys())

js
// 数组去重
let arr = [1,2,3,3,4,4,4,5]
let unique = [...new Set(arr)]
console.log(unique); // [1,2,3,3,4,4,4,5]

Map

Map对象是一个键值对的数据结构，跟对象类似，但是Map的key和value可以是任意类型.

增:map.set(key,value)

查:map.has(key) => false||true

读:map.get(key) => value

遍历：map.forEach((value, key) => console.log(key,value))

注意:当Map对象的key是直接的数组或者对象，地址指向会改变，读取不出value

js
let map = new Map([
    [1, '星期一'],
    [{ key: 2 },'星期二'],
    [[3],'星期三']
])
console.log(map.get({ key: 2 })); // undefined
console.log(map.get([3])); // undefined

模块化 #

将一组有相似功能的代码放到单独的文件中，通过模块导出，让其他文件来使用的代码，其他模块可以通过导入来使用导出的代码，这样就可以提高代码的复用性、封装性

按需导出

js
/**
 * @description
 * @param {Date} date
 * @return {*}
 */
function formatDate(date) {
    return `${date.getFullYear()}-${date.getMonth() + 1}-${date.getDate()}`
}
const DAYS_OF_YEAR = 365
export {
    formatDate,DAYS_OF_YEAR
}
